Set up your game

How to play:

  1. Enter the number of players (3-20)
  2. Add each player's name in the order they'll play
  3. Click "Start Game" to begin
  4. Pass the device to each player in order
  5. Each player clicks "Reveal" to see their role
  6. After all roles are revealed, discuss and vote!
